# Examples

This directory contains runnable example checklists for ChecklistFabrik.

```
examples/
  templates/   checklist templates (the source files you create)
  reports/     saved results from checklist runs (generated by clf-play)
```

Open the dashboard to browse all templates and reports:

```shell
clf-play
```

Or run a specific template directly:

```shell
clf-play --template templates/demo-takeoff-checklist.yml
```

For documentation on the YAML format and all available features, see the [User Guide](../docs/user_guide.md).
